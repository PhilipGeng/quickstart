## Download/Install tcloud SDK
- Download tcloud SDK \
Download latest tcloud SDK from [tags](https://github.com/turingaicloud/quickstart/tags).
- Install tcloud SDK \
Place `setup.sh` and `tcloud` in the same directory, and run `setup.sh`.
